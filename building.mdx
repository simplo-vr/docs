# How to build

## Print the frame

First step is to 3D print the main frame used for developement. You can find it [here](https://github.com/simplo-vr/SimploVR/hardware/3d/)

![frame](/images/conversion.png)

## Inserting the screen

Insert the screen on the screen hole on the front of the frame. You can use tape or something to mantain it in place for the dev phase.

![screen](/images/conversion.png)

## IMU

Place the imu in the imu case on the top of the headset frame. Pls make sure that the usb is turned to left.

![screen](/images/conversion.png)
